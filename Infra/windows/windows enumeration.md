#windows/enumeration
#### System, net and additional hostnames info:
```cmd
systeminfo
ver
hostname

ipconfig

arp /a
```
#### Current user:
```cmd
whoami

whoami /priv

whoami /groups
```
#### User and groups on host:
```cmd
net user

net group

net localgroup
```
#### Shares:
```cmd
net share

net view
```
